## ✨ Features

### 🔢 Sorting Algorithms

* Bubble Sort, Selection Sort, Insertion Sort
* Merge Sort, Quick Sort
* Animated bars with color-coded comparisons
* Live comparison & swap counter
* Adjustable speed and array size
* Pseudocode display

---

### 🔍 Binary Search

* Step-by-step execution
* Low / Mid / High pointer tracking
* Visual elimination of search space
* Decision logs
* Found / Not Found feedback

---

### 🌳 Binary Search Tree (BST)

* Dynamic node insertion
* Tree visualization
* Real-time structure updates

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript / React
* **Visualization:** DOM / Canvas / SVG
* **Styling:** CSS / Tailwind

## 📦 Installation

```bash
git clone https://github.com/your-username/dsa-visualizer.git
cd dsa-visualizer
npm install
npm start
```

## 💡 Usage

1. Select an algorithm tab
2. Adjust speed and input size
3. Click **Start Visualization**
4. Observe step-by-step execution
